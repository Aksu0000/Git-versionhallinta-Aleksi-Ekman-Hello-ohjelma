# Git-versionhallinta - Hello-ohjelma

**Kurssi:** Git-versionhallinta - SOF013AS2A-3001
**Tekijä:** Aleksi Ekman  

## Kuvaus
Tässä repositoriossa on Hello-ohjelma, jota on kehitetty kurssin harjoitusten aikana.  
Ohjelma sisältää HTML-sivun, CSS-tyylit ja JavaScriptin kellonajan näyttämiseen.  

## Sisältö
- `index.html` - ohjelman pääsivu
- `styles.css` - tyylitiedosto
- `scripts.js` - JavaScript-koodi kellonajan näyttämiseen
- Git-historian harjoitukset haarojen käytöstä ja yhdistämisestä
- lisäksi pari muuta tiedostoa, joiden avulla harjoiteltiin Git komentoja
